# OTP Space — X-проекты

Прототип раздела управления командными целями (X-проекты, индивидуальные/командные цели).

## Структура

```
otp-space/
├── index.html        # Страница X-проекта (список целей по кварталам)
├── goal.html         # Страница отдельной цели (детали + навигация по дереву)
├── css/
│   └── global.css    # Все стили: токены, layout, компоненты
├── js/
│   ├── data.js       # Общие данные: дерево целей, со-владельцы
│   ├── popup.js      # Поп-ап со-владельцев (общий между страницами)
│   ├── project.js    # Логика страницы X-проекта
│   └── goal.js       # Логика страницы цели
└── HANDOFF.md        # Дизайн-токены, описание компонентов
```

## Запуск

Открыть `index.html` в браузере. Никакой сборки не нужно.

Для удобства можно поднять локальный сервер:

```sh
python3 -m http.server 8000
# или
npx serve .
```

## Зависимости (CDN)

- Шрифт **Onest** — Google Fonts
- Иконки **Tabler Icons v2.44.0**
